# 📱 הוראות התקנת האפליקציה לשימוש אופליין

## ✅ מה השתנה?

האפליקציה עכשיו תומכת ב-**PWA (Progressive Web App)** - מה שאומר שתלמידים יכולים להתקין אותה על המחשב ולהשתמש בה **אופליין לגמרי**!

---

## 🎯 איך זה עובד?

### **עבור התלמידים:**

1. **היכנסו לאתר** דרך הדפדפן (Chrome/Edge מומלץ)
2. **בפינה העליונה של העמוד הבית** יופיע כפתור: **"💾 להורדה ושימוש אופליין"**
3. **לחצו על הכפתור** והדפדפן יבקש אישור להתקנה
4. **אשרו את ההתקנה**
5. **זהו!** עכשיו יש אייקון של האפליקציה על שולחן העבודה/תפריט התחל

### **מעכשיו:**
- פותחים את האפליקציה מהאייקון (לא צריך דפדפן!)
- עובדת **100% אופליין** - אפס צורך באינטרנט
- נראית כמו אפליקציה רגילה - אין שורת כתובת
- עדכונים אוטומטיים - כשיש אינטרנט, האפליקציה מתעדכנת לבד

---

## 🖥️ דרישות מערכת:

- **Windows 10/11** - Chrome, Edge, Opera
- **Mac** - Chrome, Safari (macOS 11.3+), Edge
- **Linux** - Chrome, Chromium, Edge

**💡 טיפ:** אם הכפתור לא מופיע, זה אומר שהדפדפן כבר מציע אוטומטית התקנה (חפשו אייקון ⊕ או ↓ בשורת הכתובת)

---

## 📋 מה נשמר במטמון אופליין?

ה-Service Worker שומר אוטומטית:
- ✅ כל קבצי ה-JavaScript וה-CSS
- ✅ כל האודיו והגרפיקה
- ✅ כל התרגילים והממשקים
- ✅ **32 קבצים** - סה"כ ~925 KB

**רק Firebase (שמירה בענן) דורשת אינטרנט!**

---

## 🚀 פריסה ל-Vercel:

כדי לפרוס את הגרסה החדשה:

```bash
# בנה את הפרויקט
npm run build

# העלה ל-Vercel (אם עדיין לא מחובר)
vercel --prod
```

או פשוט ה-push ל-GitHub יפעיל deployment אוטומטי אם הגדרת את זה!

## 🐛 Troubleshooting:

**הכפתור לא מופיע?**
- בדוק שאתה ב-Chrome/Edge מעודכן
- בדוק שהאתר מוגש דרך HTTPS (Vercel כבר מגיש ב-HTTPS)
- נסה לרענן את הדף (Ctrl+F5)

**האפליקציה לא עובדת אופליין?**
- ודא שהתקנת אותה (לא רק שמרת bookmark)
- נסה לסגור ולפתוח מחדש
- בדוק ב-DevTools (F12) → Application → Service Workers
